Proyecto académico: Web E-Commerce Complejo deportivo

Este sitio web de comercio electrónico fue desarrollado como proyecto académico en el Instituto Superior Politécnico de Córdoba. El equipo estaba formado por ocho personas, entre las que me encontraba. El sitio web está construido con las tecnologías Angular, Bootstrap, HTML, CSS, JavaScript, Django, Python y MySQL como base de datos. Cuenta con un sistema de login, registro, dashboard para el administrador y dashboard para el cliente. El administrador se puede crear a través del panel de administración de Django. La aplicación web permite hacer compras de productos y reservar instalaciones de un complejo deportivo a través de un sistema de reservas.
